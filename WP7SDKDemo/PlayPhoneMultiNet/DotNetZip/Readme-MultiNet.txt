This subdirectory contains (patched, see below) DotNetZip library sources.
Only the source files which are needed for zip-file extraction are included.
In particular, only BZip2, CommonSrc, Zip and Zlib directories are used.

DotNetZip library version : 1.9.1.8
DotNetZip library homepage: http://dotnetzip.codeplex.com/

DotNetZip-encoding.patch contains changes that were introduced to library
source to make it compatible with Windows Phone 7.
